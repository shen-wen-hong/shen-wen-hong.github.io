社群網站 -「Paka」

2021/11/13·475 字
Web PHP Vue3
Paka - 本作品屬於一個選集。
§ : Paka V2
§ : 本文
該站點使用 Vue3 框架及敝人所開發的 PHP 框架「AlpacaPHP」所架設。起初該站點僅僅是用於練習撰寫 PHP、JavaScript、Vue3 框架，及磨練後端系統設計、資料庫管理等能力，最終在朋友們的鼓勵及成就感的激勵之下，完成了該社群網站。

該站點之主要功能如同一般社群平台般，除了基本的貼文、留言、回覆留言等功能外，更多心力則是聚焦在資訊安全的部分。從用戶註冊時的 SMTP 發送驗證信件及後續 Token 驗證，到用戶登入時及登入後的一系列過程，為此也設計了許多資安子系統作為支撐。該站點的後端系統有著嚴格的驗證，任何操作都將會被驗證及紀錄。秉持資訊安全的初衷，於該站點註冊時所輸入的非必須可逆的敏感資訊(如密碼)，將會被使用 SHA256 雜湊演算法運算後存放，以確保原資料不易被洩漏及破解。

由於該專案為敝人一手打造及維護，礙於課業及升學壓力，於完成當前功能後，便停止對該專案繼續更新，轉而將心力投入在學校課程及作業、報告、專題等。

Source：https://github.com/alpaca0x0/paka/tree/v1
Demo：https://paka-v1.alpaca.tw/
